HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer element appears at the very bottom of the webpage.
  • <ul> </ul> - unordered list
  • <li> </li> - list element

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Classes always start with a '.'

Git

  • git pull origin main: pulls the main branch & updates local code
  • git push origin feature/branch-name: pushes branch to main branch
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: changes current branch to the one selected
  • git add -A: adds changes in current working branch to the staging area
  • git commit -m "Message Here": adds commit message
  • cd folder-name: changes directory to named folder
  • code . - opens IDE
